# customer_support_on_twitter_kaggle

Purpose: needs_relabel

Criterion: resolution

Nguồn: Kaggle: thoughtvector/customer-support-on-twitter

Ghi chú:
Dataset lớn về customer support trên Twitter. Cần tài khoản Kaggle và file kaggle.json. Lệnh tải: kaggle datasets download -d thoughtvector/customer-support-on-twitter

Sau khi tải về, hãy giải nén hoặc đặt dữ liệu vào folder này:

data\raw\needs_relabel\05_problem_resolution\customer_support_on_twitter_kaggle

Ví dụ nếu dùng Kaggle API:

kaggle datasets download -d thoughtvector/customer-support-on-twitter -p data\raw\needs_relabel\05_problem_resolution\customer_support_on_twitter_kaggle
unzip data\raw\needs_relabel\05_problem_resolution\customer_support_on_twitter_kaggle/customer-support-on-twitter.zip -d data\raw\needs_relabel\05_problem_resolution\customer_support_on_twitter_kaggle
